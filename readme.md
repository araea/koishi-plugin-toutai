# koishi-plugin-toutai

[![npm](https://img.shields.io/npm/v/koishi-plugin-toutai?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-toutai)

## 简介

Koishi 的投胎模拟器插件。投胎到中国或世界各地。

## 使用

1. 启动 `puppeteer` 服务。
2. 设置指令别名。
